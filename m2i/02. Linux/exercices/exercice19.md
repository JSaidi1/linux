# Exercice 19 - Chapitre

1. Créer un programme qui permet d'afficher un nombre de chapitre et de sous-partie.
2. le programme prendre l'arg1 comme le nombre de chapitre et l'arg2 comme sous-partie à afficher 

# Exemples

```bash 
./chapitre.sh 3 2
Chapitre 1
    Sous-partie 1.1
    Sous-partie 1.2
Chapitre 2
    Sous-partie 2.1
    Sous-partie 2.2
Chapitre 3
    Sous-partie 3.1
    Sous-partie 3.2
```
